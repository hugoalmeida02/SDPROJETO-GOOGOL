Sobre

O projeto Googol é um motor de pesquisa de páginas Web, desenvolvido no âmbito da disciplina de Sistemas Distribuídos do curso de Engenharia Informática da Universidade de Coimbra. O sistema tem como objetivo proporcionar funcionalidades semelhantes às de motores de busca conhecidos, como Google, Qwant e Ecosia. Inclui a indexação automática de páginas via Web crawler, a pesquisa de conteúdos com base em palavras-chave e a exibição de resultados ordenados por relevância. O sistema também permite aos utilizadores sugerirem URLs para serem indexados. A aplicação foi projetada para operar de forma eficiente e escalável, com uma interface Web que permite a consulta de páginas, análise de backlinks e visualização de estatísticas em tempo real, utilizando tecnologias como FastAPI e WebSockets.

Ajuda

Este sistema oferece várias funcionalidades interativas para melhorar a experiência do utilizador:

  • Seleção de Ação: Na página principal, pode escolher o tipo de pesquisa que deseja realizar através de um select box. As opções disponíveis são:
    • Pesquisa por palavra: Para buscar páginas que contenham palavras específicas.
    • Pesquisa de backlinks: Para consultar páginas que fazem link para uma URL específica.
    • Adicionar URL: Para sugerir um novo URL a ser indexado pelo sistema.
  • Menu e Estatísticas: O menu superior permite navegar entre várias abas, incluindo uma secção de estatísticas em tempo real, que apresenta dados atualizados automaticamente, como as 10 pesquisas mais comuns, os "Barrels" ativos e o tempo médio de resposta às pesquisas.
  • Funções de Pesquisa:
    • Página de Resultados: Ao realizar uma pesquisa, os resultados serão apresentados por paginação de 10 em 10. Além disso, duas funcionalidades adicionais estão disponíveis:
      • Indexação das "Top Stories" do Hacker News: O sistema permite indexar URLs das top stories do Hacker News que contêm os termos pesquisados.
      • Análise de Termos pela OpenAI: Após a pesquisa, a OpenAI pode ser utilizada para gerar uma análise contextual dos termos pesquisados ou das citações curtas dos resultados.
